# Changelog

All notable changes to `pg-slide-harvester` will be documented in this file.

## 0.1.0 - 2026-07-14

- Discover PostgreSQL events from official event pages.
- Download public PDF/PPT/PPTX/ODP slide assets.
- Store files by topic under `archive/<category>/`.
- Normalize filenames from talk/session titles.
- Track delayed publication and retry missing sessions.
- Generate full snapshot reports and per-run download reports.
- Record source URLs for downloaded assets.
- Provide a lightweight standard-library-only CLI.
